Projects

QuackCryption

Inspired by Rubber Duck Debugging, QuackCryption is a fully functional novel encryption method that was implemented in one weekend on 4 platforms (Desktop GUI, CLI, Website, Android). Quackcryption recognizes files encoded in the .quack format, which includes metadata about the original file, along with all of it's original data. The data is first encrypted with AES, then uses a bit-cipher to convert each bit into quacks.
